# Speed Math Quiz
A simple game which helps you develop quick basic mathematical operations skills. Compete with yourself or other players (locally) based on time! Do you think you are pro? Why not try this app? This project is made for practice!

## Required Package(s)
Python Version Used: 3.12.10

tkinter, pandas, random, fractions
```bash
pip install tkinter, pandas, random, fractions
```

## Installation Guide
Just clone this repo into your system and run the 'main.py' file then enjoy! At the end of the given time, you are displayed a leaderboard for which 'pandas' sorting function is used.
